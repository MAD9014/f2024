# JavaScript Hybrid

MAD9014 is a Hybrid course. That means that each week there is meant to be an additional hour of instructional activity with an accompanying hour of exercise work.

For this course, the Hybrid portion is a series of 10 quizzes.

Each quiz covers a different topic and is 9 random questions pulled from a pool of questions about the topic area.

The topic areas for the quizzes are:

  • JavaScript Datatypes
  • JavaScript Variables and Scope
  • JavaScript Functions
  • JavaScript Control Flow
  • JavaScript Loops
  • JavaScript Strings
  • JavaScript Numbers and Math
  • JavaScript Arrays
  • JavaScript Objects
  • JavaScript DOM

All the quizzes must be completed by the end of week 9 - November 1, 2024.

You may attempt each quiz up to three times. After each attempt you will be shown the question(s) that you got incorrect.

Only the best mark for each quiz will be kept.

The purpose of the quiz is for you to test your own knowledge and confirm that you have understood the material. You are NOT to do the quiz together with other people.

You MAY use any online resource, including video tutorials and the course notes, when completing the quiz.
