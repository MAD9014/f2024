# Complete your Project

This week, there are no more lessons. This is a project week. The PAs and the instructors will be in class just to help with final projects. All time will be devoted to answering questions about the projects.

# Looking Ahead

Next semester we will be proceeding under the assumption that you understand the material from MAD9014. You will still have this course website available to you as a reference. We are not expecting you to have memorized all the content. We just want you be able to efficiently look things up.

In MAD9022 you will be learning more about building web apps that use the React Framework. In MAD9124 you will be focused on the server-side technologies and NodeJS, learning how to build APIs.
